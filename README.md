# Аналитика бизнеса

Flask-приложение для сотрудников, которые заполняют операционные и финансовые отчеты по Ozon, WB, финансам, закупкам и логистике.

## Что есть

- регистрация и вход с указанием должности;
- SQLite-база данных;
- ручное заполнение отчетов по дням, неделям и месяцам;
- Excel-шаблоны для скачивания и обратной загрузки;
- архив всех отчетов со сворачиваемыми блоками;
- загрузка `weekly` и `monthly` HTML-дашбордов с открытием как отдельной страницы;
- переключение светлой и темной темы;
- автокаталог артикулов для повторного выбора по названию;
- автоматическая очистка отчетов и HTML-дашбордов старше 62 дней.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

Приложение запустится на `http://127.0.0.1:5000`.

## Хранение данных

- база данных хранится в `instance/business_analytics.sqlite3` и не попадает в Git;
- загруженные HTML-дашборды тоже не попадают в Git;
- Excel-шаблоны не сохраняются как файлы: после загрузки из них берутся только значения и названия артикулов.
